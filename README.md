amutake's dotfiles
==================

Installation
------------

```sh
% cd ~
% git clone git://github.com/amutake/.dotfiles.git
% cd .dotfiles
% rake
```

Rakefile
--------

```
Usage: rake <command>

Commands:
  install        Install dotfiles
  uninstall      Uninstall dotfiles
  update         Update dotfiles
  upgrade [all]  Upgrade submodules
  clean          Clean plugins
```
