This is my dotfiles.

Installation
============

```sh
% cd ~
% git clone git://github.com/amutake/.dotfiles.git
% cd .dotfiles
% ./setup.sh install
```

setup.sh
========

```
Usage: setup.sh <command>

Commands:
  help           Show this help text
  install        Install dotfiles
  uninstall      Uninstall dotfiles
  update         Update dotfiles
  clean          Clean plugins
```
